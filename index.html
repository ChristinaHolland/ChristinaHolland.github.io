Geocaching Reccommender System


I am a geocacher in my free time, and I saw a need for a recommender system to help cachers like myself chooose their next cache. This project included a supervised machine learning phase to use cache attribute data (including natural language processing and VADER sentiment analysis of text logs left at caches) to predict the popularity of newly placed caches, followed by an unsupervised machine learning phase using cosine distance and cluster analysis (KMeans and DBSCAN) to make recommendations.

Algebra Tutoring App


There are a lot of websites and apps to help students with math, but my experience in the classroom helped me see a particular need. Most sites/apps either just solve the problem for the student or require students to completely solve a problem and submit their answer before offering feedback. This is an interactive app designed to give encouragement, checks, and suggestions throughout the process of a student solving an algebra problem.
